# Sound Effect Manager
**Developed by:** Matheus Henrique Engel Stelzner  

A simple Python application that allows you to load, list, and play short sound effects through a clean graphical interface.

## Features
- Load multiple sound files (`.wav` or `.mp3`)
- Play and stop any selected sound
- Scrollable list of loaded effects
- Lightweight and easy to use

## Requirements
- Python 3.8 or higher  
- Pygame library  

## Installation
Install the required dependency before running:
```bash
pip install pygame
